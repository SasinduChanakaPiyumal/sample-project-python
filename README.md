# llm-benchmarking-py

## Usage

Build:

```shell
poetry install
```

Run Main:

```shell
poetry run main
```

Run Unit Tests:

```shell
poetry run pytest --benchmark-skip tests/
```

Run Benchmarking:

```shell
poetry run pytest --benchmark-only tests/
```
